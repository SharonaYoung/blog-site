# blog-site
Blog website template using JAM stack

## General info
This project is from DevProjects by codementor.

## Features
List of features:

* 11ty for static site generator
* A home page
* A blog page
* An About page
* Individual page for every article
* Netlify CMS
* Nunjucks templating
